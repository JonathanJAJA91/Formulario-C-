# 📋 Formulario de Usuario en C# con Avalonia

## 🚀 Descripción

Este proyecto es una aplicación de escritorio desarrollada en **C#** utilizando **Avalonia UI**, diseñada para capturar y validar datos de un usuario mediante un formulario moderno.

A diferencia de otros proyectos en C#, **NO se utilizó Windows Forms ni WPF**, ya que estos frameworks no son compatibles con Linux. En su lugar, se utilizó **Avalonia**, una tecnología multiplataforma.

---

## 🧠 Tecnologías utilizadas

- C#
- .NET 8
- Avalonia UI
- XAML
- Visual Studio Code
- Ubuntu (Linux)

---

## 🎯 Funcionalidades

✔ Formulario con 7 campos:
- Nombre  
- Apellido  
- Edad  
- Correo electrónico  
- Teléfono  
- Ciudad  
- Dirección  

✔ Validaciones:
- Campos obligatorios  
- Edad numérica  
- Teléfono solo números  
- Correo con formato válido  

✔ Interfaz:
- Diseño moderno  
- Distribución en dos columnas  
- Sin scroll  
- Resultados organizados  

✔ Visualización:
- Datos mostrados con etiquetas en **negrilla**  
- Sección final: **"Datos ingresados"**

---

## 🖥️ Interfaz

- Panel izquierdo → Formulario  
- Panel derecho → Resultados  
- Diseño limpio y centrado
